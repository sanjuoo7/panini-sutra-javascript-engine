# Sutra 1.3.84: उपाच्छ

## Overview

• Sanskrit: `उपाच्छ`  
• IAST: upāccha  
• Sense: Additionally, after the upasarga उप (upa), the root रम् takes Parasmaipada.

## Scope

- Root: रम् (ram)
- Prefix: उप (upa)
- Result: Parasmaipada designation; sets up optionality in 1.3.85 when intransitive

## Examples

- `उपरमति` → Parasmaipada
- `रमते` (no upa) → not applied
