# Sutra 1.3.81: प्राद्वहः

## Overview

**Sanskrit Text**: `प्राद्वहः`  
**Transliteration**: prādvahaḥ  
**Translation**: After प्र (pra), the root वह् (to carry) takes Parasmaipada, even when the agent benefits.
